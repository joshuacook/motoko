# Welcome to Motoko

This is an example workspace demonstrating the Context Lake pattern.

## What's here

- **tasks/** - Things to do, with status tracking
- **notes/** - Reference information and documentation
- **projects/** - Larger initiatives that group related work

## Getting started

1. Open this workspace in Claude Code: `cd workspaces/example && claude`
2. Try creating a new task: "Create a task to learn about schemas"
3. Run Tachikoma: `uv run tachikoma schema .`
